# Comunicación

Cómo nos comunicamos como equipo.

## Principios

- **Escríbelo primero.** Si lo documentas, no tienes que repetirlo. Antes de
  preguntar, busca en el handbook.
- **Asíncrono por defecto.** No todo necesita una reunión; un mensaje claro suele bastar.
- **Transparencia.** La información vive en lugares accesibles, no en cabezas o DMs.

## Canales

| Canal | Para qué |
|---|---|
| Chat del equipo | Día a día, dudas rápidas |
| Pull Requests | Cambios al handbook y al código |
| Reuniones | Solo cuando lo asíncrono no alcanza |
